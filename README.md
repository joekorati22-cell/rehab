# Rebuild Care Landing Page

A React-based landing page with multiple section variants.

## Setup

```bash
npm install
npm run dev
```

The app will start on http://localhost:3000

## Features

- Multiple hero section variants (3 options)
- Interactive hotspots on images
- Responsive design
- Smooth animations

## Switching Between Variants

Use the variant selector buttons on the right side of the screen to switch between different hero section designs.
